# Node.js server using tensorflow.js and face-api.js example
A Node.js server that analyzes face images using tensorflow.js and face-api.js libraries.

<br/>

My Blog Post: https://www.khjcode.com/post/4

<br/><br/>

## Run

<br/>

### Setup
```
npm i
```

<br/>

### Development
```
npm run dev
```

<br/>

### Build
```
npm run build
```

<br/>

### Production
```
npm run start
```
